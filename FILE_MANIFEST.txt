SETUP COMMANDS
==============
Backend:
  cd backend
  python -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  cp .env.example .env
  # Edit .env with ANTHROPIC_API_KEY
  python main.py

Frontend:
  cd frontend
  npm install
  npm run dev

Backend URL: http://localhost:8000
API Docs: http://localhost:8000/docs
Frontend URL: http://localhost:5173
